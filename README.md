# FF Sensitivity AI

Smart sensitivity recommendations for Free Fire based on device specifications, play style, and HUD layout.

## Features

- 📱 **Device Scan**: Automatically detects phone specifications
- 🎮 **HUD Analysis**: Upload screenshots to analyze button positions
- 🤖 **AI Recommendations**: Get optimized sensitivity settings
- 💾 **Save Profiles**: Store multiple sensitivity configurations
- 📊 **History Tracking**: View past recommendations
- 🌙 **Dark Mode**: Eye-friendly dark theme
- 🔄 **Sensitivity Tester**: Fine-tune with feedback
- 📤 **Share & Copy**: Share settings with friends

## Installation

### Android

1. Download the APK from [Releases](https://github.com/shahnil172011/ff-sensitivity/releases)
2. Enable "Install from Unknown Sources" in settings
3. Install the APK

### Build from Source

```bash
# Clone repository
git clone https://github.com/UnknownGuy9876/ff-sensitivity-ai.git

# Navigate to project
cd ff-sensitivity-ai

# Get dependencies
flutter pub get

# Build release APK
flutter build apk --release

# Install on device
flutter install
